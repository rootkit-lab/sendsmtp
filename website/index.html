SendSMTP

Desktop SMTP campaigns — Wails, React, and SQLite.

Import SMTPs, manage large recipient queues, personalize HTML, and send with a shared Go engine.

Install Documentation Source

What it does

  • SMTP discovery email;password or goscan blocks — MX maps to submission hosts and AUTH is verified.
  • Large lists File import avoids Wails paste limits. Deduped UNIQUE addresses; optional MX validation.
  • IMAP extract Pull contacts and credentials from INBOX/Sent, then queue and import as SMTPs.
  • Templates Placeholders, spintax {a|b|c}, and personalized {{link}}?p=<email> at send time.
